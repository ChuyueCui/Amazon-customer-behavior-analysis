# 🛒 Amazon Customer Behavior Analysis

📌 Project Overview  
This project analyzes Amazon product and customer behavior data to uncover insights on purchasing patterns, product performance, and customer sentiment. The goal is to support data-driven decision-making in pricing, marketing, and product optimization.

🔍 Key Contributions  
- Cleaned and integrated large-scale datasets (product data and customer behavior data)  
- Performed exploratory data analysis (EDA) to analyze demographics, purchase behavior, and browsing patterns  
- Conducted sentiment analysis using NLP techniques to evaluate customer feedback  
- Built visualizations to identify trends across product categories and user segments  

📊 Business Impact & Insights  
- Female users showed higher purchasing engagement across product categories  
- Lower-rated products required higher discounts, while high-rated products maintained strong demand  
- Negative feedback was mainly related to delivery delays, product quality, and packaging issues  

🏆 Results  
- Delivered actionable insights to support pricing strategy, customer targeting, and product improvement  
- Identified key drivers of customer satisfaction and dissatisfaction through sentiment analysis  

🛠️ Tools & Technologies  
- Python / R  
- Data Cleaning & EDA  
- Sentiment Analysis (NLP)  
- Data Visualization
